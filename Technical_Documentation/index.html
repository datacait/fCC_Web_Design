Introduction

CSS is a language for stylizing, arranging, and formatting websites and applications built using HTML and Java.

This highly powerful and adaptable language is responsible for creating responsive web designs.

Syntax

CSS is comprised of selectors and declarations. Selectors provide the information about which elements are being modified. Declarations dictate which modifications should be made. Properties and values are the pieces of declarations.

h1 {
font-family: sans-serif;
font-size: 16px;
}

In the code above h1 is the selector which indicates that all h1 elements should receive the declaration. The declaration indicates sans-serif for the font-family property and 16px for the font-size property.

Selectors

Selectors specify the HTML elements that are being styled. This could include the entire page, a certain section, or even a certain word on the page.

Selectors can include elements, classes, id values, etc. Multiple different elements can be combined in one selector and selectors can be designed to specify only the desired nested elements within a parent element.

Properties

Properties include an expansive list of the different things that can be styled with CSS. For example:

  • Color
  • Text
  • Display
  • Box Method (padding, spacing, etc.)
  • Layouts

Properties allow full customization of the web content.

Media

Media queries allow the web content to display according to the individual screen size of the device being used to view the site.

CSS also has features to design animations. Attributes of the animation including length of animation, timing of animation, and actions within the animation can all be modified.
